✅ 한글/영어/특수문자 완벽 지원
✅ 엑셀, 구글 시트에서 저장한 CSV 자동 인코딩 감지
✅ UTF-8, EUC-KR 인코딩 자동 처리

🔗 구글 시트 연동 방법

1. 웹에 게시하기:
• 구글 시트 열기 → 파일 → 공유 → 웹에 게시
• 형식: CSV 선택 (중요!)
• 게시 버튼 클릭

2. URL 복사:
• 생성된 URL 전체를 복사
• 또는 시트 편집 URL의 /d/ 다음 ID만 복사

3. 암기장에 입력:
• 위 입력창에 붙여넣기
• 카테고리 이름 입력
• "구글 시트에서 불러오기" 클릭

⚠️ 문자 깨짐 해결법

한글이 깨져 보이는 경우:
• 구글 시트에서 한글 입력 후 저장했는지 확인
• 웹에 게시 시 형식을 반드시 CSV로 선택
• 게시 후 1-2분 대기 (구글 서버 반영 시간)

엑셀에서 CSV 저장 시:
• 파일 → 다른 이름으로 저장 → CSV UTF-8 (쉼표로 분리) 선택

여전히 깨진다면:
• 구글 시트 → 파일 → 다운로드 → CSV
• 다운로드한 파일을 직접 업로드
